# Simulador de Resiliencia Hotelera 🏨

**Herramienta interactiva** que simula la gestión de crisis en hoteles mediante un *"journey"* basado en hitos. Desarrollado para:

- Capacitar equipos en protocolos de recuperación  
- Visualizar el impacto de decisiones estratégicas  
- Mejorar la experiencia del cliente durante fallas operativas  

## Características clave ✨
- 🎮 **Interfaz tipo "elige tu propia aventura"** con 4 hitos críticos  
- 📊 **Métricas visuales** (colores/sonidos reflejan el estado de recuperación)  
- 🛠 **Base técnica simple**: HTML + Vanilla JS + JSON  
- 📱 **Responsive**: Funciona en móviles y desktop  

## Uso rápido 🚀
1. Clona el repo  
2. Abre `index.html` en tu navegador  
3. Explora los hitos:  
   - **Falla crítica** → Estrategias → Recuperación parcial → Recuperación total  

## Personalización 🛠
Edita `datos.json` para:  
- Cambiar narrativas/colores (`color: "#FFEBEE"`)  
- Añadir nuevos hitos  
- Modificar opciones de decisión
